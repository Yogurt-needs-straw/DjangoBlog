# DjangoBlog

## 1.博客系统（案例）

开发一个博客系统，包含：博客列表、详细、登录、注册、**评论、点赞、发布博客**。

### 1.2功能

- 博客列表

  ```
  路由 + 视图（时间倒序排序） + 序列化
  ```

- 博客详细 

  ```
  路由（PK） + 新视图或原视图 + 序列化
  ```

- 评论列表

  ```
  在URL上通过GET方式传入博客ID，根据博客ID获取相关评论。
  
  提示：请求可以单独，也可以在博客详细的请求中返回。
  ```

- 注册

  ```
  输入用户信息+重复密码进行注册
  ```

- 登录

  ```
  登录成功，生成TOKEN+失效日期，返回。
  ```


- 评论



none
